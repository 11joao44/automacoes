------------------------------ Raspagem de Dados ------------------------------

• Instalação: no terminal > `pip install requests`
• Instalação: no terminal > `pip install beautifulsoup4`
• Instalação: no terminal > `pip install pip install openpyxl`
• Documentação: http://www.crummy.com/software/BeautifulSoup/bs4/doc/

-------------------------------------------------------------------------------

--------------------------------- Requests ------------------------------------

Requests é uma biblioteca usada para fazer solicitações HTTP 
para interagir com recursos na web, como servidores da web, 
APIs e outros serviços da internet.

-------------------------------------------------------------------------------

------------------------------ Beautiful Soup ---------------------------------

usada para analisar documentos HTML e XML, 
permitindo que você extraia informações
páginas da web e faça a análise de dados.

-------------------------------------------------------------------------------

--------------------------------- Openpyxl ------------------------------------

permite que você leia dados de planilhas XLSX existentes
criar novas planilhas a partir do zero. pode acessar células específicas 
em uma planilha para ler ou escrever dados nelas.
